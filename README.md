# TaCZ Enchantments Overhaul

A Minecraft Forge mod that adds new enchantments to enhance the Timeless and Classic Zero (TaCZ) gun system. This mod provides additional gameplay options through customizable enchantments that can be balanced to your liking.

## New Enchantments

This mod adds 7 enchantments for TaCZ guns with configurable parameters:

- **Bullseye**: Increases headshot damage by 50% per level
- **Deep Impact**: Increases armor penetration by 5% per level  
- **Firepower**: Increases bullet damage by 10% per level
- **Precision**: Increases accuracy by 5% per level
- **Punch Through**: Increases bullet penetration through entities by +1 per level
- **Capacity**: Increases magazine capacity by 50% per level
- **Bullet Retention**: Chance to not consume bullets when firing (10% per level)

All enchantment effects can be configured in the config which is accessible in-game via the mod settings menu, or
by altering the `tacz_enchantments_overhaul.json5` file in the `config` folder. Additionally, the enchantment level
spans can be configured allowing for more fine-tuned balancing of when and where the enchantments are available.

## Enchantment Blacklisting

The mod allows you to blacklist specific guns for specified enchantments. This means you can have certain 
enchantments not be available for specific guns, by default all enchantments are available for all guns. 
This is also configurable in the config file.

## Compatibility

This mod is designed to work with:
- [Timeless and Classic Zero](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classics-zero) 1.1.6+
- [TACZ Tweaks](https://www.curseforge.com/minecraft/mc-mods/tacz-tweaks)
- [TACZ Durability](https://www.curseforge.com/minecraft/mc-mods/tacz-durability)
- [Shots Fired](https://www.curseforge.com/minecraft/mc-mods/shots-fired)
